### 3. 게임모드

  1. 조준영 모드

    * 문장이 주어진다.

    * 버튼을 눌러 문장을 녹음한다.

    * 녹음본을 텍스트로 변환해서 체크

    * 걸린 시간 체크

    * 정답률을 넘긴 것 중에서 시간순으로 순위 지정

    * 미리 정한 라운드(3,5,7) 수 만큼 누적 시간을 매겨 순위 지정
